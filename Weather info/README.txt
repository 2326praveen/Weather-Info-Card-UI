
Project: Multi-City Responsive Weather UI


Description:
A modern and responsive weather UI that displays weather cards for multiple cities. This project is created using only pure HTML5 and CSS3. It demonstrates proficiency in building card-based interfaces, responsive CSS Grid layouts, and modern styling techniques like "glassmorphism" over a background image.

---

Features:
- Pure HTML5 and CSS3. No JavaScript.
- Fully responsive design that works on desktop, tablet, and mobile devices.
- A CSS Grid layout that automatically adjusts the number of columns based on screen size.
- A full-screen background image.
- Modern "frosted glass" UI style on the cards using backdrop-filter.
- Use of SVG icons for crisp rendering on all screen resolutions.
- Well-structured and commented code for easy understanding.
